# Account-Manager
This is an python based gui application made with tkinter module and sqlite3

## Requirement

> Python 3,
> Windows, Linux, Mac

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install

```bash
pip install tkinter
pip install PIL
pip install functools
pip install sqlite3
```
